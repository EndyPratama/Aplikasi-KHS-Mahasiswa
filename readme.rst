- Admin
- User
- Student
- Lecturer
- Matkul
- Transcript
- Message
- Class.

The user visits the website and login, the user will be directed to a page that matches the user's authorization level.

If the user is a student then what can be accessed are:

- Biodata

- Account settings
- Go out
